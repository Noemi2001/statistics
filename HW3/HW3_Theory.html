Exercise 2: Recall briefly the definition and math notions relevant to "probability space" and make some simple examples, indicating among the triple of the space the meaning of each element in your particular example. If you wanted to model probabilistically the homework Exercise 1, explain what are the 3 sets of your probability space and their elements, in this case.

    In probability theory, a probability space or a probability triple (Ω, F, P) is a mathematical construct that provides a formal model of a random process or "experiment". A probability space consists of three elements:
    1. Sample Space (Ω): The sample space represents the set of all possible outcomes of a random experiment or process.
    2. Event space (F): is a set of events. It is a collection of subsets of the sample space that defines which sets of events are considered "measurable" or "events of interest."
    3. Probability Function (P): The probability measure assigns a real number in the range [0, 1] to each event in the event space.

    Example: Suppose you are modeling the outcome of rolling a fair six-sided die. In this case, the probability space is as follows:
    1. Sample Space (Ω): The sample space represents all possible outcomes when rolling the die. Ω = {1, 2, 3, 4, 5, 6}, as these are the possible results when you roll a six-sided die.
    2. Event space (F): In this simple example, the sigma algebra could be all possible subsets of Ω, including the empty set and the sample space itself. Σ = {∅, {1}, {2}, {3}, {4}, {5}, {6}, {1, 2}, {1, 3}, ..., {1, 2, 3, 4, 5, 6}}.
    3. Probability Function (P): For a fair die, the probability of each outcome is 1/6, as each face has an equal chance of landing face up. Therefore, P({1}) = P({2}) = P({3}) = P({4}) = P({5}) = P({6}) = 1/6.
