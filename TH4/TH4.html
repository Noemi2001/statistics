Thesis 4: Glivenko–Cantelli theorem

The Glivenko–Cantelli theorem is also called the Fundamental Theorem of Statistics and it is an important result in probability theory and mathematical statistics. It provides a convergence result for the empirical distribution function toward the true distribution function. The theorem is named after Russian mathematicians Dmitrii Alekseevich Glivenko and Francesco Paolo Cantelli, who independently proved it in the 1930s.

The theorem states that the empirical distribution function (EDF) of a random sample converges uniformly to the true distribution function. Formally, if \(X_1, X_2, \ldots, X_n\) are independent and identically distributed random variables with cumulative distribution function (CDF) \(F(x)\), and \(F_n(x)\) is the empirical distribution function based on the sample, then the Glivenko–Cantelli theorem asserts that:
\[ \sup_{x \in \mathbb{R}} |F_n(x) - F(x)| \rightarrow 0 \text{ almost surely as } n \rightarrow \infty. \] This convergence is uniform, meaning that the maximum difference between the empirical and true distribution functions tends to zero almost surely.

So, the Glivenko-Cantelli theorem is crucial because it affirms that through sampling, we can effectively capture the characteristics dictated by the distribution function \( F \). As the sample size grows, the empirical distribution function converges uniformly to the true distribution function, indicating that the probabilities inferred from samples become more accurate with increased data availability. Consequently, studying distributions through samples is a valid and reliable approach for learning and making inferences about the underlying distribution.

Proof Outline:

Let's consider the case of a continuous random variable \(X\). We fix a partition of the real line \(-\infty = x_0 < x_1 < \cdots < x_{m-1} < x_m = \infty\) such that \[F(x_j) - F(x_{j-1}) = \frac{1}{m}\] for \(j = 1, \ldots, m\). Here, \(F(x)\) is the cumulative distribution function (CDF). Now, for any \(x \in \mathbb{R}\), there exists a unique \(j \in \{1, \ldots, m\}\) such that \(x \in [x_{j-1}, x_j]\). It follows that: \[ \begin{aligned} F_{n}(x)-F(x) &\leq F_{n}(x_{j})-F(x_{j-1}) = F_{n}(x_{j})-F(x_{j})+\frac{1}{m}, \\ F_{n}(x)-F(x) &\geq F_{n}(x_{j-1})-F(x_{j}) = F_{n}(x_{j-1})-F(x_{j-1})-\frac{1}{m}. \end{aligned} \] Rearranging the inequalities, we get: \[\|F_{n}-F\|_{\infty }=\sup _{x\in \mathbb {R} }|F_{n}(x)-F(x)|\leq \max _{j\in \{1,\dots ,m\}}|F_{n}(x_{j})-F(x_{j})|+{\frac {1}{m}}.\] Now, since \(\max_{j \in \{1, \ldots, m\}} |F_n(x_j) - F(x_j)| \to 0\) almost surely (by the strong law of large numbers), we can ensure that for any positive \(\epsilon\) and any integer \(m\) such that \(1/m < \epsilon\), there exists an \(N\) such that for all \(n \geq N\): \[\max_{j \in \{1, \ldots, m\}} |F_n(x_j) - F(x_j)| \leq \epsilon - \frac{1}{m}\] Combining this with the previous result, we can conclude that \(\|F_n - F\|_{\infty} \leq \epsilon\) almost surely. This fulfills the definition of almost sure convergence.
